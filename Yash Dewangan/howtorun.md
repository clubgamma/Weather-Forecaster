<h1 align="center">How to run</h1>
<br>
1-> Clone the branch repo
<br>
2-> add the API key and update the Weather.py
<br>
3-> run the Following command in terminal
<br>      
        pip install -r requirements.txt
        <br>
        set FLASK_APP=Weather
        <br>
        flask run
        <br>
